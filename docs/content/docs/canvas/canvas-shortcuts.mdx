# Canvas Shortcuts

This document covers common mouse and keyboard operations on the canvas.

## View

- Drag canvas blank area: Pan the view.
- Mouse scroll wheel: Zoom the canvas.
- Zoom slider: Fine-tune zoom.
- Reset view button: Return to default zoom and center.

## Selection

- `Ctrl / Cmd` + Drag: Box-select multiple nodes.
- `Shift / Ctrl / Cmd` + Click node: Add to or remove from selection.
- `Ctrl / Cmd` + `A`: Select all canvas nodes.
- `Esc`: Deselect and close the current popover.

## Editing

- `Ctrl / Cmd` + `C`: Copy selected nodes.
- `Ctrl / Cmd` + `V`: Paste nodes.
- `Delete / Backspace`: Delete selected nodes or connections.
- `Ctrl / Cmd` + `Z`: Undo.
- `Ctrl / Cmd` + `Shift` + `Z`: Redo.
- `Ctrl / Cmd` + `Y`: Redo.

## Images and Assets

- Drag in image files: Upload images to canvas.
- Import image button: Select images from local files.
- My Assets: Select an asset and insert it onto the canvas.

## Undo and Redo Scope

Undo and redo track canvas node, connection, viewport, background mode, and assistant session changes. Canvas project names, account status, global AI configuration, and "My Assets" save operations are not part of the current canvas history.
