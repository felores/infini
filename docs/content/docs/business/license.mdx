## Project Intent

Infinite Canvas remains open source to let more people view and modify the code, avoid being locked into specific API channels, connect their own APIs and models, and use the canvas to organize prompts, reference images, assets, and generation results to build their own AI creation workflows.

## Usage Recommendations

If you are only using it locally for learning, research, or personal use, you can use it under the open source license directly.

If you plan to use the project for commercial products, private delivery, SaaS services, or long-term internal team systems, please first confirm whether your usage meets the open source requirements of AGPL-3.0.

If your use case cannot continue to open source modified code, or requires closed-source delivery, commercial licensing, private deployment support, or custom development services, see [Business Cooperation](/docs/business/business).

## What is AGPL-3.0

The core requirements of AGPL-3.0 can be summarized as:

- **Open source obligation**: If you use Infinite Canvas's AGPL code, regardless of how you or your downstream users use, modify, further develop, or distribute it, you must fully publish the final code derived from this project. This means publishing all the code, not just the modified parts, and rewriting it in a different framework does not exempt you from the original code's obligations.
- **License continuity**: Derivative code based on this project must continue to be open-sourced under AGPL-3.0. You cannot switch to another license or make this portion of the code closed-source.
- **Network services must also be open source**: Even if you only deploy a modified version as a website, SaaS, or other network service, as long as others can use it over the network, you must provide the corresponding source code to those users and continue to comply with AGPL-3.0.
- **Preserve copyright notices**: You cannot remove the original project author information, copyright notices, license statements, or source attribution. Users must know where the code comes from and what license it follows.
- **No additional restrictions**: You cannot add extra restrictions to AGPL code derived from this project, such as prohibiting others from redistributing the code or requiring others to purchase a license, service, or product before using this open source code.
- **Disclaimer**: The project is provided as-is. The author does not guarantee the code is bug-free and is not responsible for any results, losses, or risks arising from using this project.

This project prohibits closed-source commercial use. If you wish to use Infinite Canvas in a commercial project, please respect open source, strictly follow the AGPL-3.0 license, continue to open source the corresponding code, and give back to the open source community.
