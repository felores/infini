# Sponsorship

If you want to get sponsored visibility in the project docs, README, or related pages, contact the project maintainer directly by email to discuss placement, duration, and display format.

## Suitable Sponsorship Content

- AI tools, model services, API services, or developer tools.
- Products related to image, video, and text generation workflows.
- Services suitable for open source project users and AI creators.

## Recommended Information to Provide

- Brand or product name.
- Desired placement and duration.
- Links, copy, and assets to display.
- Budget range and cooperation method.
