# Render Deploy

Click the link below to deploy to Render:

[Deploy to Render](https://render.com/deploy?repo=https://github.com/basketikun/infinite-canvas)

## Deployment Steps

1. Click `Deploy to Render`.
2. Log in to Render and connect your GitHub account as prompted.
3. Confirm deployment.

After deployment completes, open the `.onrender.com` domain assigned by Render.

## Free Tier Notes

Uses Render's free Web Service by default:

- Sleeps after ~15 minutes of idle; auto-wakes on next visit.
- Main app data is stored in the browser by default; third-party prompts are fetched directly by the browser frontend.
- Suitable for testing and demos, not for long-term data storage.

For long-term use, prefer deploying to Vercel or configuring WebDAV sync for browser-local data.
