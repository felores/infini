## Funciones Principales

- **Lienzo Infinito**: proyectos con mГәltiples lienzos, arrastrar y soltar nodos, conexiones, minimapa, deshacer/rehacer, importar/exportar.
- **CreaciГіn con IA**: llamadas directas desde el navegador a endpoints compatibles con OpenAI. Soporta texto-a-imagen, imagen-a-imagen, ediciГіn de imГЎgenes de referencia, preguntas y respuestas de texto, generaciГіn de audio y video; Seedance 2.0 disponible mediante Huoshan Ark Agent Plan.
- **Asistente de Lienzo**: conversa con el asistente sobre los nodos seleccionados y sus nodos upstream, genera imГЎgenes e inserta los resultados de vuelta en el lienzo.
- **Agente Local**: conecta Codex / Claude Code a travГ©s del Canvas Agent local mediante MCP para operar el lienzo actual.
- **Plugin Codex App**: instala el plugin de la app Codex para registrar MCP automГЎticamente e intentar iniciar el agente local.
- **Biblioteca de Prompts**: acceso directo desde el navegador a mГәltiples proyectos open-source de GitHub, almacenados en cachГ© en IndexedDB.

DocumentaciГіn completa en [Funciones](docs/content/docs/overview/features.mdx).

## Inicio RГЎpido

La API Key, Base URL, lienzos, materiales e historial de generaciГіn se almacenan localmente en el navegador por defecto.

### Desarrollo Local

```bash
git clone git@github.com:basketikun/infinite-canvas.git
cd infinite-canvas
cd web
bun install
bun run dev
```

### Docker

```bash
git clone git@github.com:basketikun/infinite-canvas.git
cd infinite-canvas
docker compose up -d
```

Puerto predeterminado: 3000 вҖ” visita `http://localhost:3000`.

En el primer inicio, abre la configuraciГіn en la esquina superior derecha e ingresa tu `Base URL` y `API Key` compatibles con OpenAI.

## ConfiguraciГіn AutomГЎtica con New API

Si usas New API, completa lo siguiente en `ConfiguraciГіn del Sistema вҶ’ Chat вҶ’ Agregar ConfiguraciГіn de Chat`:

```text

```

Al navegar a esta URL se abrirГЎ automГЎticamente el diГЎlogo de configuraciГіn con la API Key y Base URL pre-cargadas.
Si tienes tu propio despliegue, reemplaza `https://canvas.best` con tu direcciГіn.
